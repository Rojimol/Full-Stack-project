# Travel Bridge ✈️🌍

## 🚀 Overview
Travel Bridge is a full-stack travel management web application designed to simplify trip planning by integrating flight search, hotel booking, and itinerary management into a single platform.

This project demonstrates end-to-end full stack development using modern technologies with a focus on performance, scalability, and user experience.

---

## 🛠️ Tech Stack

### Frontend
- React.js
- HTML5, CSS3, JavaScript

### Backend
- Java
- Spring Boot (REST APIs)

### Database
- MySQL

### Tools
- Git & GitHub
- Postman (API testing)
- VS Code / Eclipse

---

## ✨ Features

- 🔍 Flight search functionality  
- 🏨 Hotel booking system  
- 📅 Travel itinerary management  
- 🔐 Secure user authentication & authorization  
- 📊 User dashboard for trip tracking  
- 💰 Expense tracking for better travel budgeting  
- 📱 Responsive UI for seamless experience across devices  

---

## 🧩 Architecture

- RESTful APIs built with Spring Boot  
- Frontend communicates with backend via HTTP APIs  
- MySQL database for persistent data storage  
- Modular and scalable project structure
